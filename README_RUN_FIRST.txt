VAT Spreadsheet Preparation Prototype
Run This Demo First

1. Unzip the full demo folder before you try to run it.
2. Open the unzipped folder.
3. Double-click VAT_Spreadsheet_Demo.exe.
4. The local browser UI should open automatically.
5. If you want a quick sample file, use data\demo\sample_data.csv inside the package.

If the browser does not open automatically, check the terminal window and visit the local address shown there, usually:
http://127.0.0.1:7860

Notes:
- The prototype runs locally on your machine.
- The packaged demo is the main Windows delivery entry for teachers and reviewers.
- The local Automatic Explanation works without internet access.
- AI Suggestions are optional and depend on provider settings, API key availability, and network conditions.
- Output files for each run are written locally under the output folder created beside the executable.
