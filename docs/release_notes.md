## v0.3.0 - Dissertation Demo Refinement

### Title

`v0.3.0 - Dissertation Demo Refinement`

### Summary

This release marks the current dissertation demo milestone for the AI-Assisted VAT Spreadsheet Review Prototype. It strengthens the evaluation story, expands review-signal coverage, refines exporter and dashboard behavior, and improves the Downloads experience so outputs are easier for non-technical reviewers to interpret.

### Highlights

- Added contextual anomaly detection support.
- Added controlled VAT evaluation testbed generation and related evaluation artefacts.
- Improved exporter and review dashboard behavior.
- Refined the Downloads tab with clearer recommended exports and a plain-language overview panel.
- Reduced avoidable false positives through optional VAT code validation fixes.

### Suitable for

- Dissertation demonstrations
- Supervisor or assessor walkthroughs
- Current packaging and documentation baseline
